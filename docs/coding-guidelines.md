# Coding Guidelines

## Style Guide

This project follows the PEP 8 Python Style Guide.

---

## Naming

Variables

```python
student_name
```

Functions

```python
calculate_total()
```

Classes

```python
StudentManager
```

Constants

```python
MAX_USERS
```

---

## Function Design

Functions should:

- Have one responsibility
- Be reusable
- Include docstrings
- Return meaningful values

---

## Comments

Use comments only when necessary.

Prefer descriptive variable and function names over excessive comments.

---

## Error Handling

Always validate user input.

Handle exceptions gracefully.

Never allow unexpected crashes.

---

## Formatting

Maximum line length:

```
88 characters
```

Use Black for formatting whenever possible.
