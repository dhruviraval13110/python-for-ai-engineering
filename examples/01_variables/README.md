## Interview Questions

### What is a variable?

A variable is a named location in memory that stores data.

### Difference between int and float?

- int → whole numbers
- float → decimal numbers

### What does input() return?

Always a string.

### Why use type conversion?

To convert data from one type to another.

Example:

```python
age = int(input("Age: "))
```
