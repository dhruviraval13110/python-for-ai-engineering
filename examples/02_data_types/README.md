Python provides several built-in data types for storing and manipulating data.

## Learning Objectives

After completing this section, you will be able to:

- Differentiate between mutable and immutable data types.
- Choose the appropriate data structure for a problem.
- Perform common operations on Python collections.
- Use `type()` and `isinstance()` for type checking.

## Interview Questions

### What is the difference between a list and a tuple?

- **List:** Mutable (can be modified).
- **Tuple:** Immutable (cannot be modified after creation).

### Why use a set?

A set stores unique values and supports efficient membership testing.

### What is a dictionary?

A dictionary stores data as key-value pairs.

### What is the difference between `type()` and `isinstance()`?

- `type()` returns the exact type of an object.
- `isinstance()` checks whether an object is an instance of a class or its subclasses.
