## Interview Questions

### What is the difference between `/` and `//`?

- `/` performs floating-point division.
- `//` performs floor division.

### What is the difference between `==` and `is`?

- `==` compares values.
- `is` compares object identity (whether two references point to the same object).

### When are bitwise operators used?

Bitwise operators manipulate individual bits and are commonly used in low-level programming, optimization, permissions, and flags.

### What is operator precedence?

Operator precedence determines the order in which operations are evaluated in an expression.
