# Contributing

Thank you for your interest in contributing to this project.

## Ways to Contribute

You can contribute by:

- Reporting bugs
- Suggesting new features
- Improving documentation
- Optimizing existing code
- Adding new Python examples
- Improving code quality
- Writing tests

---

## Development Setup

Clone the repository:

```bash
git clone https://github.com/dhruviraval13110/python-for-ai-engineering.git
```

Navigate into the project:

```bash
cd python-for-ai-engineering
```

Create a virtual environment:

```bash
python -m venv .venv
```

Activate it.

Windows

```bash
.venv\Scripts\activate
```

Linux/macOS

```bash
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Coding Standards

Please follow:

- PEP 8
- Meaningful variable names
- Modular design
- Clear comments when necessary
- Keep functions small and reusable

---

## Commit Message Style

Examples:

```
feat: add expense tracker example

fix: resolve division by zero bug

docs: improve README

refactor: simplify utility functions

test: add unit tests
```

---

## Pull Requests

Before submitting:

- Ensure code runs correctly
- Update documentation if needed
- Keep pull requests focused
- Follow the repository structure

Thank you for helping improve this project.
