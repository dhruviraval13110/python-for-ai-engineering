# 🏠 Rent Calculator

A simple Python program that helps split rent and expenses between multiple people easily.

## 📌 Features

* Calculate total rent per person
* Add food expenses
* Include electricity cost
* Simple and easy to use

## 🚀 How to Run

1. Install Python on your system
2. Download or clone this repository
3. Run the file:

```bash
python rent_calculator.py
```

## 💡 Example

Enter your hostel/flat rent, food expenses, electricity units, and number of people — the program will calculate how much each person needs to pay.
